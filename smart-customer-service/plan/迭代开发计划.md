**状态**: ⏸️ 待开始

**验收标准**:

- 分块策略可配置(按字符数/段落/标题)
- 分块后保留元数据(来源文件、页码、标题层级)


```python
@dataclass
```

- 使用 `PyPDF2` 解析PDF, `python-docx` 解析Word

### 子任务 RAG-03: 混合检索策略实现

**状态**: ⏸️ 待开始  
**任务描述**: 实现向量检索 + BM25关键词检索 + 重排序的混合检索 pipeline

**验收标准**:
- 支持纯向量检索、纯关键词检索、混合检索三种模式
- 混合检索相关性评分比单一检索提升 > 15%
- 支持动态调整权重(向量权重α, 关键词权重β)
- 实现 Maximal Marginal Relevance (MMR) 去重

**执行步骤**:

#### 1. [Design] 需求与设计 (0.5天)

**检索接口**:
```python
def hybrid_search(
    query: str,
    k: int = 5,
    mode: str = "hybrid",  # vector | keyword | hybrid
    alpha: float = 0.7,           # 向量权重
    beta: float = 0.3,            # 关键词权重
    use_mmr: bool = True,
    mmr_lambda: float = 0.5
) -> List[SearchResult]
```

**算法设计**:
- **BM25实现**: 使用 `rank-bm25` 库计算关键词得分
- **分数归一化**: Min-Max归一化到 [0,1] 区间
- **加权融合**: `final_score = α * vector_score + β * bm25_score`
- **MMR去重**: 平衡相关性和多样性

**Langfuse埋点**:
```python
with create_span("hybrid_retrieval"):
    - create_span("vector_search")
    - create_span("bm25_search")
    - create_span("score_fusion")
    - create_span("mmr_reranking")
    - add_event("retrieval_results", output_data={"count": k, "mode": mode})

score_trace("retrieval_relevance", avg_relevance)
score_trace("retrieval_diversity", diversity_score)
```
