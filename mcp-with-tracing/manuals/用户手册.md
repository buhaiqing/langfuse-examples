## 快速开始

### 1. 安装依赖

```bash
# 创建虚拟环境
uv venv
source .venv/bin/activate

# 安装依赖
uv pip install -r requirements.txt
```

### 2. 配置环境变量

编辑 `.env` 文件：

```bash
# Langfuse API 配置
LANGFUSE_PUBLIC_KEY=pk-lf-你的公钥
LANGFUSE_SECRET_KEY=sk-lf-你的密钥
LANGFUSE_HOST=https://cloud.langfuse.com

# 企业微信机器人 (可选)
WECOM_WEBHOOK_URL=https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=你的 key
```

### 3. 启动服务

```bash
python src/server.py
```

### 4. 验证安装

```bash
# 运行测试
python scripts/test_langfuse_connection.py
```

看到 "✓ TEST PASSED" 表示安装成功！

## API 使用指南

### 基础追踪

```python
from src.observability import (
    init_observability,
    ObservabilityConfig,
    observe_tool,
)

# 初始化
config = ObservabilityConfig()
init_observability(config)

# 使用装饰器
@observe_tool(name="my_tool")
def my_function(arg1, arg2):
    return result
```

### 会话管理

```python
from src.observability import set_session, get_session_id

# 开始新会话
set_session(
    session_id="user-123-session",
    user_id="user-123"
)

# 获取当前会话 ID
session_id = get_session_id()
```

### 反馈收集

```python
from src.observability.feedback import (
    record_acceptance,
    record_rejection,
    record_rating,
)

# 记录接受
record_acceptance(
    trace_id="trace-123",
    user_id="user-123",
    comment="Great response!"
)

# 记录拒绝
record_rejection(
    trace_id="trace-123",
    reason="inaccurate",
    comment="Wrong information"

# 记录评分 (1-5)
record_rating(
    trace_id="trace-123",
    rating=4,
    comment="Very good"
)
```

### 告警配置

#### 方式 1: 使用 YAML 配置文件（推荐）

```yaml
# config/alerts.yaml
rules:
  - name: high-error-rate
    metric: error_rate
    threshold: 0.05
    operator: gt
    severity: warning
    cooldown_minutes: 30
    max_alerts_per_hour: 5
```

服务器启动时会自动加载此配置文件。

#### 方式 2: 使用 Python API

```python
from src.observability.alerting import (
    get_alert_manager,
    AlertSeverity,
    AlertRule,
)
from src.observability.notifiers import WeComNotifier

manager = get_alert_manager()

# 配置企业微信通知
manager.register_notification_handler(
    "wecom",
    WeComNotifier("你的 webhook URL")
)

# 配置告警规则
manager.create_rule(
    name="success-rate-low",
    metric="success_rate",
    threshold=0.95,
    operator="lt",
    severity=AlertSeverity.WARNING,
    cooldown_minutes=30,  # 冷却期，防止告警风暴
    max_alerts_per_hour=5  # 每小时最大告警数
)
```

#### 方式 3: 启动后台自动巡检

```python
from src.observability.alert_monitor import start_alert_monitor

# 启动告警监控（每 5 分钟检测一次）
monitor = start_alert_monitor(check_interval_minutes=5)
```

#### 方式 4: 启用智能 ML 异常检测

```python
from src.observability.smart_alerting import SmartAlertManager

# 启动 ML 异常检测（每 10 分钟检测一次）
smart_manager = SmartAlertManager(detection_interval_minutes=10)
smart_manager.start_monitoring()

# 自动检测:
# - Prophet 时间序列异常（单指标）
# - PyOD 多维异常检测（多指标关联）
```

## 告警配置

### 告警级别

| 级别 | 响应时间 | 通知方式 |
|------|----------|----------|
| INFO | 4 小时 | 仅记录 |
| WARNING | 30 分钟 | 企业微信 |
| CRITICAL | 5 分钟 | 企业微信 + 邮件 |

### 常见告警类型

#### 1. 成功率告警

```python
# 当成功率低于 95% 时告警
manager.create_rule(
    name="success-rate-low",
    metric="success_rate",
    threshold=0.95,
    operator="lt",
    severity=AlertSeverity.WARNING,
    cooldown_minutes=30,  # 冷却期
    max_alerts_per_hour=5  # 限流
)
```

#### 2. 延迟告警

```python
# 当 P95 延迟超过 500ms 时告警
manager.create_rule(
    name="latency-high",
    metric="latency_p95_ms",
    threshold=500,
    operator="gt",
    severity=AlertSeverity.CRITICAL,
    cooldown_minutes=15,
    max_alerts_per_hour=10
)
```

## 常见问题

### Q1: 如何查看追踪数据？

访问 Langfuse 控制台：https://cloud.langfuse.com

### Q2: 如何分析用户行为？

使用会话 ID 过滤：
1. 在 Langfuse 中选择 Sessions
2. 输入 session_id
3. 查看完整用户旅程

### Q3: 如何比较 Prompt 版本？

使用版本过滤：
1. 在 Langfuse 中创建 Dashboard
2. 添加过滤器：metadata.prompt_version
3. 对比不同版本的指标

### Q4: 告警不触发怎么办？

检查清单：
- [ ] API keys 配置正确
- [ ] 网络连接正常
- [ ] 告警阈值设置合理
- [ ] 通知渠道已注册

### Q5: 如何自定义告警规则？

```python
manager.create_rule(
    name="custom-alert",
    metric="your_metric",
    threshold=100,
    operator="gt",  # 可选：gt, lt, gte, lte, eq
    severity=AlertSeverity.WARNING,
    window_minutes=60,  # 时间窗口
    cooldown_minutes=30,  # 冷却期
    max_alerts_per_hour=5  # 限流
)
```

### Q6: 如何启用智能 ML 异常检测？

1. 安装 ML 依赖：
```bash
uv pip install prophet pyod pandas numpy scikit-learn
```

2. 启动智能检测：
```python
from src.observability.smart_alerting import SmartAlertManager

manager = SmartAlertManager(detection_interval_minutes=10)
manager.start_monitoring()
```

3. 查看文档：[智能告警快速启动](../docs/smart-alerting-guide.md)
