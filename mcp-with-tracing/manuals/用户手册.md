## 快速开始

### 1. 安装依赖

```bash
# 创建虚拟环境
uv venv
source .venv/bin/activate

# 安装依赖
uv pip install -r requirements.txt
```

### 2. 配置环境变量

编辑 `.env` 文件：

```bash
# Langfuse API 配置
LANGFUSE_PUBLIC_KEY=pk-lf-你的公钥
LANGFUSE_SECRET_KEY=sk-lf-你的密钥
LANGFUSE_HOST=https://cloud.langfuse.com

# 企业微信机器人 (可选)
WECOM_WEBHOOK_URL=https://qyapi.weixin.qq.com/cgi-bin/webhook/send?key=你的 key
```

### 3. 启动服务

```bash
python src/server.py
```

### 4. 验证安装

```bash
# 运行测试
python scripts/test_langfuse_connection.py
```

看到 "✓ TEST PASSED" 表示安装成功！

## API 使用指南

### 基础追踪

```python
from src.observability import (
    init_observability,
    ObservabilityConfig,
    observe_tool,
)

# 初始化
config = ObservabilityConfig()
init_observability(config)

# 使用装饰器
@observe_tool(name="my_tool")
def my_function(arg1, arg2):
    return result
```

### 会话管理

```python
from src.observability import set_session, get_session_id

# 开始新会话
set_session(
    session_id="user-123-session",
    user_id="user-123"
)

# 获取当前会话 ID
session_id = get_session_id()
```

### 反馈收集

```python
from src.observability.feedback import (
    record_acceptance,
    record_rejection,
    record_rating,
)

# 记录接受
record_acceptance(
    trace_id="trace-123",
    user_id="user-123",
    comment="Great response!"
)

# 记录拒绝
record_rejection(
    trace_id="trace-123",
    reason="inaccurate",
    comment="Wrong information"
)

# 记录评分 (1-5)
record_rating(
    trace_id="trace-123",
    rating=4,
    comment="Very good"
)
```

### 告警配置

```python
from src.observability.alerting import (
    get_alert_manager,
    AlertSeverity,
    AlertRule,
)
from src.observability.notifiers import WeComNotifier

manager = get_alert_manager()

# 配置企业微信通知
manager.register_notification_handler(
    AlertChannel.WEBHOOK,
    WeComNotifier("你的 webhook URL")
)

# 配置告警规则
manager.register_rule(AlertRule(
    name="success-rate-low",
    metric="success_rate",
    threshold=0.95,
    operator="lt",
    severity=AlertSeverity.WARNING,
))
```

## 告警配置

### 告警级别

| 级别 | 响应时间 | 通知方式 |
|------|----------|----------|
| INFO | 4 小时 | 仅记录 |
| WARNING | 30 分钟 | 企业微信 |
| CRITICAL | 5 分钟 | 企业微信 + 邮件 |

### 常见告警类型

#### 1. 成功率告警

```python
# 当成功率低于 95% 时告警
AlertRule(
    name="success-rate-low",
    metric="success_rate",
    threshold=0.95,
    operator="lt",
    severity=AlertSeverity.WARNING,
)
```

#### 2. 延迟告警

```python
# 当 P95 延迟超过 500ms 时告警
AlertRule(
    name="latency-high",
    metric="latency_p95_ms",
    threshold=500,
    operator="gt",
    severity=AlertSeverity.CRITICAL,
)
```

## 常见问题

### Q1: 如何查看追踪数据？

访问 Langfuse 控制台：https://cloud.langfuse.com

### Q2: 如何分析用户行为？

使用会话 ID 过滤：
1. 在 Langfuse 中选择 Sessions
2. 输入 session_id
3. 查看完整用户旅程

### Q3: 如何比较 Prompt 版本？

使用版本过滤：
1. 在 Langfuse 中创建 Dashboard
2. 添加过滤器：metadata.prompt_version
3. 对比不同版本的指标

### Q4: 告警不触发怎么办？

检查清单：
- [ ] API keys 配置正确
- [ ] 网络连接正常
- [ ] 告警阈值设置合理
- [ ] 通知渠道已注册

### Q5: 如何自定义告警规则？

```python
manager.register_rule(AlertRule(
    name="custom-alert",
    metric="your_metric",
    threshold=100,
    operator="gt",  # 可选：gt, lt, gte, lte, eq
    severity=AlertSeverity.WARNING,
    window_minutes=60,  # 时间窗口
))
```
