# 快速入门指南

> 5 分钟快速上手 MCP Langfuse 可观测平台

---

## 1. 安装 (2 分钟)

```bash
# 克隆项目
cd /Users/bohaiqing/test/ai_agent/langfuse_senarios/example2

# 创建虚拟环境
uv venv
source .venv/bin/activate

# 安装依赖
uv pip install -r requirements.txt
```

---

## 2. 配置 (1 分钟)

### 2.1 获取 Langfuse Keys

1. 访问 https://cloud.langfuse.com
2. 登录/注册
3. 创建项目
4. 复制 API Keys

### 2.2 配置 .env 文件

```bash
cp .env.example .env
vim .env
```

填入你的 Keys:
```
LANGFUSE_PUBLIC_KEY=pk-lf-你的公钥
LANGFUSE_SECRET_KEY=sk-lf-你的密钥
```

---

## 3. 测试 (1 分钟)

```bash
# 运行连接测试
python scripts/test_langfuse_connection.py
```

看到以下输出表示成功:
```
✓ Langfuse client initialized successfully
✓ TEST PASSED
```

---

## 4. 开始使用 (1 分钟)

### 方式 1: 使用装饰器

```python
from src.observability import observe_tool, init_observability

# 初始化
init_observability()

# 使用装饰器
@observe_tool(name="my_tool")
def my_function(arg1, arg2):
    return result
```

### 方式 2: 使用会话追踪

```python
from src.observability import set_session

# 开始会话
set_session(
    session_id="user-123-session",
    user_id="user-123"
)
```

### 方式 3: 收集反馈

```python
from src.observability.feedback import record_acceptance

# 用户接受
record_acceptance(
    trace_id="trace-id",
    comment="Great!"
)
```
